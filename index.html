Comandos de voz

Grabar: para iniciar a capturar la información, notara además que una señal se pondra de color rojo en la parte inferior derecha

Pausa: para detener la captura de la información, notara además que una señal se pondra de color gris en la parte inferior derecha

Siguiente: para pasar al campo siguiente

Anterior: para pasar al campo anterior

Corregir: para eliminar la ultima palabra que se dijo

Código: para entrar al modo código, notara además que una señal se pondra de color rojo en la parte inferior derecha

Normal: para detener el modo código, notara además que una señal se pondra de color gris en la parte inferior derecha

Eliminar: para limpiar un campo del formulario

Nuevo: para guardar los campos del formulario en el csv y limpiarlo para recibir nuevos datos

Descargar: para descargar el formulario en formato csv
